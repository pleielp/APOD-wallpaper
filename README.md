## 설치 가이드

APOD_wallpaper.py 파일을 받아 시작 애플리케이션으로 등록한다.

바탕화면으로 설정된 사진파일은 ~/wallpaper 폴더에 저장된다.

- 버전 정보
    - OS: Linux Mint 19.1 Cinnamon
    - python: 3.7, 3.6
    - pygame: 1.9.4
